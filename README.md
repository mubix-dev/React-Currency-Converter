# 💱 Currency Converter (React)

A modern and responsive Currency Converter web app built using **React**, **custom hooks**, and real-time exchange rate API.

---

## 🚀 Features

* 🔄 Convert between multiple currencies in real-time
* 🔁 Swap currencies instantly
* ⚡ Auto update conversion on input change
* 📡 Fetch live exchange rates using API
* 🧩 Reusable components (`InputBox`)
* 🧠 Custom Hook (`useCurrency`) for API logic
* 🎨 Clean UI with Tailwind CSS

---

## 🛠️ Tech Stack

* **React.js**
* **JavaScript (ES6+)**
* **Tailwind CSS**
* **Currency API** (Exchange rates)

---

---

## ⚙️ How It Works

1. User enters an amount
2. Selects "From" currency
3. Selects "To" currency
4. App fetches live exchange rates
5. Displays converted amount instantly

---

## 🔧 Installation

```bash
# Clone the repository
git clone https://github.com/mubix-dev/React-Currency-Converter.git

# Navigate into the folder
cd currency-converter

# Install dependencies
npm install

# Run the app
npm run dev
```
